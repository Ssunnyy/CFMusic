# CFMusic
使用FreeStreamer流媒体库写的一个音乐播放器，实现的功能有歌词显示，锁屏控制，上一曲，下一曲，暂停，进度条拖拽播放，滑动切换等。目前还会不定时更新一些新功能上来，欢迎交流。
